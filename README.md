# Book Snippet App

Note taking app for your books. Search for a book, add it to your list of books, then write snippet of notes about each book.

## Installation
- `bundle install`
- `yarn install`
- `rails db:migrate`
- `rails server`

## Gems
- `normalize-rails`
- `webpacker`
- `jquery-rails`
- `momentjs-rails`
- `will_paginate`

## Packages
- `google-books-search`
- `stimulus`
